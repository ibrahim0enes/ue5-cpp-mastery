# XP Terminal

A single self-contained `index.html` — no build step, no dependencies. Open it in any browser (double-click `index.html`, or serve the folder) to check off sujet exercises and watch your XP/rank update.

- Mirrors the exact exercise list and XP values in `docs/sujets/`
- Bonus rows stay locked until every mandatory exercise in that project is checked, same as 42's rule
- Progress is saved in your browser's local storage — always open it from the same browser/device, or it'll look reset
- Each project card has a "Print / Save as PDF" button that regenerates that project's checklist as a clean printable page

This is a personal tracker, not a synced or hosted service — it's just a static file that lives in the repo with everything else.
