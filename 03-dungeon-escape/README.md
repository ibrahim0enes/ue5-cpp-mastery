# 02 · Dungeon Escape

A first-person escape room. Goal: a pressure plate, a lock, and a collectible-item system, all built from composable Actor Components talking to each other through delegates instead of hard references.

- **Lectures:** 50 · 6h 50m
- **Sujet:** [`docs/sujets/Sujet - Bölüm 3 - Dungeon Escape.pdf`](../docs/sujets)
- **XP:** 150 mandatory (+35 bonus)
- **`.uproject`:** `DungeonEscape.uproject` (create it directly in this folder)

## Core ideas

Inheritance vs. composition · pointers & references · delegates · Enhanced Input · line/shape tracing · `TArray`
